Per eseguire l'Admin-Manager:
	-eseguire un doppio click sull'iconta
	-oppure da riga di comando (dopo aver ragiunto la cartella) java -jar AdminManager-EmotionalSongs.jar

Test:
Effettuati test su i seguenti sitemi operativi:
	- MacOS Ventura 
	- Windows 10
	- Windows 11

Bug:
Su sistemi operativi basati Apple, il jar non sembre carica le immagini correttamente
